# Random Number Sign Checker

## Description
This Python script generates a random integer between **-10** and **10** and determines whether the number is:

- Positive
- Negative
- Zero

The result is then displayed on the screen.

## Requirements
- Python 3

## Usage

1. Save the script as `0-positive_or_negative.py`.
2. Run the script:

```bash
python3 0-positive_or_negative.py
